# Code of Conduct

## Our standard

Be respectful, direct, and constructive.

Expected behavior:

- Focus feedback on ideas and implementation details.
- Assume good intent and communicate clearly.
- Avoid harassment, insults, or discriminatory language.

Unacceptable behavior:

- Personal attacks
- Threats or intimidation
- Deliberate disruption/spam

## Enforcement

Project maintainers may remove comments/issues/PRs that violate this code and, if needed, restrict participation.
